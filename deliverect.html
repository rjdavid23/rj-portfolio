Overview

At Deliverect, I led the redesign of the settings experience used by restaurant managers to configure channels and POS integrations. The existing system was difficult to navigate and lacked clear structure.

Problem

  • Disorganized settings layout across multiple tabs
  • Excessive scrolling to find basic options
  • Difficulty locating configuration settings
  • Cognitive overload managing multiple integrations

Solution

  • Introduced structured navigation using clear tabs
  • Reduced reliance on long scrolling pages
  • Improved grouping and hierarchy of settings
  • Designed clearer configuration flows
